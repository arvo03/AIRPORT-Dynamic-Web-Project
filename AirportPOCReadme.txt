ProjName - AirportPOC
Java - JDK11
Tested on Apache Tomcat Server 9.0


REQUIREMENT NOTES FOR WINDOWS- 
Should have a CLASSPATH system variable with location of these - 
1. JDK/bin
2. .
3. tomcat's "servlet-api.jar" location
4. your local AirportPOC codebase directory 

Just for reference my CLASSPATH variable for example looked like this - 
C:\Program Files\Java\jdk-11.0.11\bin\;.;C:\Program Files\Apache Software Foundation\Tomcat 9.0\lib\servlet-api.jar;D:\Study\CodeBase\AirportPOC
